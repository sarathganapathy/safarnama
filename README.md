# Safarnama

A Full MERN Stack Safarnama Application.

## Quick Start

#### Setup
```
npm install mongodb

#### for Development

Start the database
```
mongod

```

Start the client
```
npm run dev
```

Start the server
```
npm start
```

#### for Production

Start the database
```
mongod

```

Start the Server
```
npm run prod
```

#### Other Commands

```
npm run build
npm run lint
npm run lint:fix
```
